# TikOmni Skills

中文 | [English (README.md)](./README.md)

这个仓库提供给 AI Agent 使用的 TikOmni Skill 包。

## 用户只需要知道这些

- 该 Skill 让 Agent 通过 TikOmni API 获取多平台结构化数据。
- 内置固定提取流程（如抖音/小红书主页提取）和通用路由能力。
- 普通用户**不需要**运行仓库里的维护/调试脚本。

## 使用入口

- 官网：https://tikomni.com
- 控制台：https://app.tikomni.com
- API：https://api.tikomni.com
- API 文档：https://docs.tikomni.com

## 基本使用方式

请在你的 AI Agent 环境中使用该 Skill（如 OpenClaw/Codex/Claude Code 集成）。

如果需要安装或集成帮助，请在本仓库提 issue。

## 安全说明

不要把任何密钥或私有凭据提交到 Git 仓库。
