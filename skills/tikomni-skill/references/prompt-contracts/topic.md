# Prompt Contract · 选题（topic）

## Inputs
- `title`（作品标题，必填优先）
- `asr_clean`（清洗后文案，优先于 raw）
- `category`（可选，已有分类）
- `tags`（可选）

## Method (stable)
1. 按关键词命中做**基础类型三分**：流量型 / 人设型 / 营销型。  
2. 结合标题+ASR做**细分主题词提取**（Top 3）。  
3. 从问题词/痛点词提取**受众痛点**。  
4. 输出必须给出命中证据，不得仅主观判断。

## Output Format
- `基础类型`：`<类型>（命中信号 N 个）`
- `细分主题`：`词1、词2、词3`
- `受众痛点`：`痛点1、痛点2`

## Constraints
- 只使用输入数据推断，不引用外部知识。
- 结论必须可追溯到词命中或句段。
- 不输出表格。

## Fallback / Data-insufficient
- `title` 与 `asr_clean` 均缺失：输出“数据不足”。
- 主题词命中为 0：输出“数据不足（未检测到显著主题词）”。
- 痛点词命中为 0：输出“以降低门槛/快速落地为主（显性痛点词不足）”。
