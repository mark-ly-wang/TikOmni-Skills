# 自定义扩展指南

## 0. 先配置再扩展

1. 在 `.env` 中配置运行环境（`TIKOMNI_API_KEY` 为必填）。
2. 编辑 `references/runtime-config.zh-CN.md`，定义输出目录、markdown 默认值和执行策略。
3. 若用户在提示词指定了自定义配置路径，应优先读取该路径。

## 1. 可扩展项

1. 新 intent 路由规则。
2. 字段清洗映射。
3. Markdown 模板。
4. 新的固定 playbook。

## 2. 新能力必备定义

每个新能力都必须明确：

1. 输入 schema。
2. 主/备接口链路。
3. fallback 条件与上限。
4. markdown 输出模板。

## 3. 不可破坏规则

1. 抖音文案提取固定走 U2。
2. 小红书文案提取固定“字幕优先，U2 兜底”。
3. 命中主页提取后，禁止切回自由路由。
