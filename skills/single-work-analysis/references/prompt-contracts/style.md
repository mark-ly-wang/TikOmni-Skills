# Prompt Contract · 文风（style）

## Inputs
- `title`
- `asr_raw`（主字段）
- `asr_clean`（遗留兼容字段）

## Method (stable)
1. 分句后统计平均句长（短/中/长）。
2. 统计疑问/感叹/陈述分布。
3. 统计人设代词（我/你/我们/大家）与修辞触发词（不是/而是/其实/必须）。
4. 输出“人设+句式+语气”三层结论。

## Output Format
- `句式结构`：`<类型>，平均句长约 X 字`
- `语气分布`：`疑问 A / 感叹 B / 陈述 C`
- `人设与修辞`：`人设代词命中 N 次，强调/转折词命中 M 次`

## Constraints
- 至少 2 个统计指标；禁止空泛描述。
- 仅基于文本统计，不猜测作者背景。
- 不输出表格。

## Fallback / Data-insufficient
- `asr_raw`（主字段）
- `asr_clean`（遗留兼容字段） 为空：三项均输出“数据不足”。
- 分句失败：标注“需人工复核”。
