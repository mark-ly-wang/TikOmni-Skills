# 卡片路由与目录口径（Topic 35）

本文件定义对标学习目录中的三类卡片落盘规则。

## 1) 目录口径（固定）

- 单作品卡：`10-内容系统/15-对标研究/01-作品对标卡/`
- 作者主页样本集：`10-内容系统/15-对标研究/02-作者样本集/<platform>-<author>/`
- 作者对标卡（分析结论）：`10-内容系统/15-对标研究/03-作者对标卡/`
- `04-对标结论/` 先保留，不在本轮改造范围。

## 2) card_type 约定

- `work` -> `01-作品对标卡`
- `author_sample_work` -> `02-作者样本集/<platform>-<author>/`
- `author` -> `03-作者对标卡`

兼容别名（自动归一）：

- `sample_work`
- `homepage_sample`

## 3) 命名与格式

- `work` 文件名前缀：`CBV-`
- `author` 文件名前缀：`CBA-`
- `author_sample_work` 文件名前缀：`CBV-`（与单作品卡同模板）
- 作者主页样本卡与单作品卡正文结构必须一致。

## 4) 结构硬约束

frontmatter 至少包含：

- `duration_ms`（视频时长，毫秒）

作品卡正文必须包含：

1. 选题
2. 文风
3. Hook
4. 结构
5. CTA
6. 好在哪里
7. 差在哪里
8. 改写

并包含 ASR 双轨：

- `ASR_CLEAN`（正文，清洗版；按固定规则分句分段：每句单行、每段 2-4 句）
- `ASR_RAW`（附录，原始转写）
